# 05.01 Ticket Detail

> **Feature Area:** [[../README|05-Ticket-Detail]]
> **Feature ID:** 05.01

## Behavior

### Layout
Full page replacement of main content area (no panels, no drawers). Navigation via back button or `Escape`.

### Sections (top to bottom)
1. **Header bar:** Back arrow, ticket ID (mono), action buttons (Close/Reopen, Edit)
2. **Title:** Large heading, full text
3. **Metadata row:** Status dot+label, Priority dot+label, Type badge, Created date
4. **Tags:** Pill list, clickable (filters list view to that tag)
5. **Dependencies:** Linked ticket IDs, clickable (navigate to that ticket)
6. **Divider**
7. **Body:** Full rendered markdown (react-markdown + remark-gfm)

### Markdown Rendering
- GFM tables, checkboxes, code blocks with syntax highlighting
- Checkboxes are interactive — click toggles them, writes back to file via API
- Headings get anchor links
- Code blocks use Geist Mono, Zinc-800 background

### Actions
- **Close/Reopen** button — toggles ticket status
- **Edit** — opens raw markdown in a simple textarea overlay (not a full editor)
- **Priority cycling** — click priority dot cycles through levels
- **Escape** — returns to previous view (list or board), preserving scroll position
