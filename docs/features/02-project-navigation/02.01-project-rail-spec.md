# 02.01 Project Rail

> **Feature Area:** [[../README|02-Project-Navigation]]
> **Feature ID:** 02.01

## Behavior

### Layout
Narrow vertical strip (48-56px) at the left edge. Each project rendered as a rounded square (40px) with the project's first letter in Geist font. Active project has a blue left-border pill indicator (3px wide, 60% height, rounded, like Discord).

### Scenarios
- Click project icon → switch to that project, load its tickets
- Hover icon → tooltip with full project name (150ms delay)
- Keyboard `1`-`9` → jump to project by position
- Keyboard `H`/`L` → previous/next project
- Bottom of rail: settings gear icon

### Visual States
- **Default:** Zinc-700 letter on Zinc-800 background, rounded-xl
- **Hover:** Zinc-600 background, slight scale (1.05) via CSS transition
- **Active:** Blue-500 left pill, Zinc-700 background, white letter
- **Loading:** Pulse animation on the active icon while tickets load

## Decisions

### ADR-1: Discord model over Linear sidebar
- **Status:** Accepted
- **Context:** Linear uses a full sidebar with project tree. Discord uses a narrow icon rail.
- **Decision:** Icon rail — saves horizontal space, faster switching, works for 4-8 projects.
- **Consequences:** No room for project tree/hierarchy. Fine for our scale.
