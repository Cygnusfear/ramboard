# 03.01 List View

> **Feature Area:** [[../README|03-List-View]]
> **Feature ID:** 03.01

## Behavior

### Layout
Full-width table. Each row is a ticket. Default sort: priority (urgent first), then created date (newest first).

### Row Contents
- **Checkbox** — hidden by default, visible on hover or when selection mode active
- **Ticket ID** — mono font, muted (Zinc-500)
- **Title** — primary text, truncated with ellipsis, full on hover
- **Tags** — small pills below title, max 3 visible + overflow count
- **Priority** — colored dot + label (far right area)
- **Status** — colored dot + label
- **Created** — relative date ("3d ago"), muted

### Interactions
- Click row → navigate to ticket detail (full page)
- Click checkbox → toggle selection (does NOT navigate)
- Column headers → click to sort (toggle asc/desc)
- Hover row → subtle Zinc-800/50 background highlight

### Sorting
Sortable by: priority, status, created date, title (alphabetical). Active sort shown with arrow indicator in column header.

### Filtering
Header bar filters: status dropdown, priority dropdown, tag dropdown. Active filters shown as removable chips. Filters are AND-combined.
